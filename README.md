# Temperature Converter

A simple Python project for converting temperatures between Celsius, Fahrenheit, and Kelvin.

## Features

- Celsius to Fahrenheit
- Fahrenheit to Celsius
- Celsius to Kelvin
- Kelvin to Celsius
- Fahrenheit to Kelvin
- Kelvin to Fahrenheit
- Convert a list of Celsius temperatures

## Run

```bash
python temperature_converter.py
```
